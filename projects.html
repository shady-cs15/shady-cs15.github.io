Visual Odometry

Developed visual odometry software for the Karlsruhe drive sequences. It currently uses disparity maps for estimating relative scale between two consecutive poses, thereby minimizing drift. More details mentioned in blog post.

Stereo Vision Library

Developed this simple library on top of OpenCV for depth reconstruction from a stereo rig. Contains separate modules for stereo calibration, rectification, reprojection. More details can be found in the blog post.

Wear teleoperation of ArDrones

Worked with JdeRobot in Google Summer of Code 2015 to develop a distributed architecture for remote teleoperation of ArDrones from android wear. The visual feed was further processed using Gaussian Mixture Models (GMMs) to obtain foreground mask of moving vehicles and their traffic density.

Crosslingual Parsing

As research interns under Prof. Sudeshna Sarkar we explored recursive neural nets for crosslingual parsing. Our pipeline used a recursive neural network based shift reduce parser (Manning et al.) and a word vector translation matrix learnt from a small bilingual dictionary. A brief summary of the project is given in the report attached.
